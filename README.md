# سیستم مدیریت شهریه مجتمع آموزشی

## ✨ ویژگی‌های اصلی
- ثبت‌نام دانش‌آموز جدید
- ویرایش و حذف اطلاعات دانش‌آموز
- جستجوی دانش‌آموزان بر اساس نام، کد ملی و پایه تحصیلی
- تغییر وضعیت تحصیلی (فعال/غیرفعال/انصرافی)
- تعریف روش‌های پرداخت (نقدی/کارت به کارت/واریز بانکی/چک)
- تعریف برنامه قسطی برای هر دانش‌آموز
- ثبت پرداخت جدید با تولید رسید و شماره پیگیری یکتا
- محاسبه خودکار جریمه تأخیر
- مشاهده صورتحساب و تراکنش‌های دانش‌آموز
- مشاهده لیست بدهکاران با امکان مرتب‌سازی
- دریافت خروجی PDF و Excel از گزارش‌ها
- داشبورد مدیریت مالی با آمار کلیدی

## 👥 بازیگران (Actors)
- **مدیر مجتمع**: مدیریت دانش‌آموزان و وضعیت تحصیلی
- **مدیر مالی**: مدیریت پرداخت‌ها، اقساط، گزارش‌ها و داشبورد
- **سیستم**: تولید رسید و محاسبه جریمه به صورت خودکار
